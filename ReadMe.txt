WhatsApp Group Automation
This script automates the process of sending messages to WhatsApp group using Selenium and Python. It opens WhatsApp Web, locates the desired group, and sends a specified message to group.

Features:
Uses Selenium and Python to automate browser actions
Supports headless mode for running the script without a visible browser window
Handles the login process by scanning the QR code manually
Install the necessary dependencies: Selenium, ChromeDriver.
Set the path to the ChromeDriver executable in the code (chrome_driver_path variable).
Replace the group_names list with the names of the WhatsApp groups you want to target.
Replace the message variable with your desired message.
Run the script and follow the instructions to scan the QR code and start the automation process.
